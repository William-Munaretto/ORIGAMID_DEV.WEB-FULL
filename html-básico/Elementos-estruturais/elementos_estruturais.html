SPAN

A tag "span" é um elemento estrutural que não possui qualquer significado semântico. Ela serve para estruturarmos o conteúdo através do CSS. Ela possui características de um elemento de linha.

SECTION E ARTICLE

Section = Representa uma parte genérica do documento html.
Article = Representa uma parte independente do documento.

Obs.:Section e Article são elementos semânticos que servem para estrutura o documento.

HEADER E FOOTER

Header = Esse é o cabeçalho da página ou artigo.

Conteúdo do artigo

Footer = Esse é o rodapé da página ou artigo.

Obs.: "header" e "footer" são elementos semânticos que servem para estruturar o documento. Existe ainda a tag "nav" para a parte de navegação do site.
