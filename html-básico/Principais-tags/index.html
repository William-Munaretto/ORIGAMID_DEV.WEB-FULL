  • "src" = source

UL (Unordered List) - Lista não ordenada.

A "ul"cria uma lista de itens não ordenados. Cada item da lista deve estar dentro de uma "li" (list item). Podem existir listas dentro de listas.

Exemplo:

Numerais:

  • Um
  • dois
  • três

OL (Ordered List) - Lista ordenada.

Diferente da "ul", a "ol" é uma lista ordenada. Serão atribuídos numerais automaticamente aos itens da lista.

Exemplo:

Classificação do campeonato:

  1. William;
  2. Ricardo;
  3. João.

Significados das tags:

  • "tr" = Table row = linha.
  • "th" = Table head = cabeçalho.
  • "td" = Table data = dado da tabela.

As colunas de uma tabela no HTML são identificadas através da "th", ou seja, a primeira "th" é a primeira coluna, a segunda "th" segunda coluna, e assim sucessivamente. No restantnte, cada "td" dentro de cada "tr" segue a identação e a mesma lógica para completar os dados nas linhas, ou seja, a primeira "td" na primeira coluna, a segunda "td na segunda coluna, todas dentro da mesma "tr" (linha). Tal situação pode ser verificada no exemplo da tabela acima.
