COMO CRIAR UM DOCUMENTO CSS:

  • Abra o editor de texto e crie um novo documento
  • Salve e nomeie ele de nomedocumento.css
  • link o documento ao arquivo html

Outras características:

  • Alguns estilos são aplicados aos elementos filhos. (exemplo: se for colocar no "body" uma cor azul, os elementos que não tiverem formatação de CSS, receberão automaticamente a cor azul tbem.)
  • o que vem por último tem mais força.(se for definida por primeiro uma cor azul e por ultimo uma cor vermelha para o mesmo elemento, prevalecerá vermelho.)
